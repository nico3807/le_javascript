2. Les Instructions

Description & Syntaxe

Le code se lit comme une recette de cuisine : ligne par ligne, de haut en bas. Chaque ligne terminée par un ; est une instruction.

Exemple : Lancement d'un jeu

// Étape 1 : Le programme dit bonjour
alert("Bienvenue dans le jeu Super MMI !");

// Étape 2 : On affiche un message technique dans la console (invisible pour le joueur)
console.log("Chargement des graphismes...");

// Étape 3 : On initialise le score du joueur
let score = 0;

// Étape 4 : On prévient que c'est prêt
alert("Prêt ? C'est parti !");
