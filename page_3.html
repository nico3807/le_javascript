3. Les Calculs

Description & Syntaxe

En programmation, et surtout dans le jeu vidéo, les maths sont partout. On utilise les opérateurs arithmétiques pour faire évoluer les données :

  • + : Addition (Gagner des points, ajouter de la vie)
  • - : Soustraction (Perdre des points, subir des dégâts)
  • * : Multiplication (Bonus combo, expérience doublée)
  • / : Division (Partager un butin, calculer une moyenne)

Exemple : Gestion du Score d'un Joueur

let score = 0; // Le joueur commence à 0
let pointsEnnemi = 50;
let multiplicateurCombo = 2;

// 1. Le joueur élimine un ennemi (+50 points)
score = score + pointsEnnemi; 
// Le score est maintenant de 50

// 2. Le joueur active un bonus "Double Score" (*2)
score = score * multiplicateurCombo; 
// Le score passe à 100 (50 x 2)

// 3. Le joueur tombe dans un piège et perd des points (-20)
score = score - 20; 
// Le score final est de 80

console.log("Score final : " + score + " pts");
