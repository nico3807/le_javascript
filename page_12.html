12. Fonctions avec Arguments

Description & Syntaxe

Une fonction est encore plus utile si on peut lui donner des informations (arguments) pour qu'elle travaille dessus.

Exemple : Convertisseur Euros vers Dollars

// La fonction attend un montant en euros (variable 'euros')
function convertirEnDollars(euros) {
    let tauxChange = 1.1; // 1 euro = 1.1 dollar
    let resultat = euros * tauxChange;
    return resultat; // La fonction "renvoie" le calcul
}

// Utilisation :
let prixEnDollars = convertirEnDollars(20); 
console.log("20 euros valent environ " + prixEnDollars + " dollars.");

console.log("50 euros valent " + convertirEnDollars(50) + " dollars.");
