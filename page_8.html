8. La Boucle while

Description & Syntaxe

On utilise while quand on ne sait pas combien de temps ça va durer, mais qu'on a une condition d'arrêt.

Exemple : Simulation de téléchargement

let pourcentage = 0;

// Tant que le téléchargement n'est pas fini (moins de 100%)
while (pourcentage < 100) {
    console.log("Téléchargement : " + pourcentage + "%");
    
    // On augmente le pourcentage (simulé ici par +10)
    pourcentage = pourcentage + 10;
}

console.log("Téléchargement terminé ✅");
