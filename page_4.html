4. Conditions (If/Else)

Description & Syntaxe

Le "If" (Si) permet à votre programme de prendre des décisions intelligentes en fonction de l'état des données.

Exemple : Affichage d'un niveau de batterie

let batterie = 15; // Pourcentage de batterie
if (batterie > 50) {
    // Si batterie supérieure à 50%
    console.log("✅ Tout va bien.");
} 
else if (batterie > 20) {
    // Sinon, si elle est entre 20 et 50%
    console.log("⚡Batterie moyenne.");
} 
else {
    // Dans tous les autres cas (donc moins de 20%)
    console.log("❌ Alerte ! Branchez le chargeur !");
}
