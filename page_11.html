11. Fonctions Simples

Description & Syntaxe

Une fonction est comme un "mini-programme" stocké quelque part. Il ne s'exécute pas tout seul : il attend qu'on l'appelle.

Exemple : Afficher une notification

// 1. Déclaration : On prépare la fonction
function afficherErreur() {
    alert("❌ Une erreur est survenue !");
    console.log("Rapport d'erreur envoyé au serveur...");
}

// 2. Appel : Pour l'instant rien ne s'est passé.
// On décide maintenant de déclencher la fonction :
afficherErreur();
