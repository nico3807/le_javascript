6. Opérateurs logiques

Description & Syntaxe

Parfois, une seule condition ne suffit pas. On doit combiner plusieurs critères.

Exemple : Ouvrir un coffre fort

let aLaCle = true;
let estProcheDuCoffre = true;
let estEnCombat = false;

// && (ET) : Il faut avoir la clé ET être proche
if (aLaCle && estProcheDuCoffre) {
    console.log("Vous insérez la clé...");
}

// ! (NON) : Il ne faut PAS être en combat pour ouvrir le menu
if (!estEnCombat) {
    console.log("Ouverture de l'inventaire possible.");
}

// || (OU) : Si j'ai la clé OU si je suis un voleur (crochetage)
let competenceVoleur = false;
if (aLaCle || competenceVoleur) {
    console.log("Le coffre s'ouvre !");
}
