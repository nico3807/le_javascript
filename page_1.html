1. Les Variables

Description & Syntaxe

Imaginez un profil utilisateur. Certaines données ne changeront jamais (date de naissance), d'autres changeront souvent (nombre de likes).

  • const : Pour ce qui ne bouge pas (Date de naissance, Nom de serveur).
  • let : Pour ce qui évolue (Compteur, Pseudo, Panier).

Exemple : Profil Utilisateur

// Je déclare le nom du site (ne changera pas)
const nomSite = "MySocialNetwork";

// Je déclare le pseudo de l'utilisateur (peut changer)
let pseudo = "DarkSasuke99";

// Je déclare son nombre d'amis (va changer)
let nombreAmis = 150;

// Plus tard, l'utilisateur change de pseudo et ajoute un ami :
pseudo = "LightNaruto01"; 
nombreAmis = 151;

// nomSite = "AutreSite"; // ❌ Erreur ! Impossible de modifier une const
