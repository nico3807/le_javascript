9. La Boucle do while

Description & Syntaxe

Le do...while est rare mais utile : il garantit que le code s'exécute au moins une fois, même si la condition est fausse dès le début.

Exemple : Lancer de dé

let resultatDe;

do {
    // On simule un lancer de dé (nombre aléatoire entre 1 et 6)
    resultatDe = Math.floor(Math.random() * 6) + 1;
    console.log("J'ai lancé un : " + resultatDe);
    
} while (resultatDe !== 6); // On recommence tant qu'on n'a pas fait 6

console.log("Enfin un 6 ! Je peux sortir mon pion.");
