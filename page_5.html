5. Opérateurs de comparaison

Description & Syntaxe

Attention à la différence entre le double égal == (laxiste) et le triple égal === (strict). En JS, on préfère presque toujours le triple égal !

Exemple : Vérification code PIN

let codeSaisi = "1234"; // L'utilisateur tape souvent du texte
let codeSecret = 1234;  // Le code enregistré est un nombre

// Comparaison laxiste (==)
if (codeSaisi == codeSecret) {
   console.log("Ouverture autorisée (converti auto)"); 
   // JS transforme le texte "1234" en nombre 1234
}

// Comparaison stricte (===) - RECOMMANDÉ
if (codeSaisi === codeSecret) {
   console.log("Ceci ne s'affichera pas car les types sont différents");
}
